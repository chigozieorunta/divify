The Divify plugin is a simple Divi plugin containing list of custom modules for extending your WordPress Divi sites.

== Description ==

The Divify plugin helps you extend the functionality of your Divi WordPress site by adding custom modules that cater for virtually any kind of design problem you may come across in your WordPress website.

At a glance, this plugin helps you:

* Add custom modules to your Divi builder.

= How do I use it? =
